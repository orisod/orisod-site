What does this tool do?

This tool combines one or more JPG images into a single PDF document, one image per page, in the order you select them. It's a quick way to turn photos, scanned pages, or receipts into a single shareable, printable file.

When should you use it?

  • You have several photos of a document and need one combined file
  • You want a format that prints and displays consistently on any device
  • A form or application requires a PDF instead of separate image files
  • You need to archive scanned receipts or pages as a single document

Why use Orisod for this?

The conversion happens entirely in your browser using jsPDF — your images are never uploaded to a server. You can add multiple images, reorder or remove them before converting, and each page automatically matches the image's orientation.

Frequently asked questions

Can I combine multiple JPGs into one PDF?
Yes, add as many images as you need — each becomes its own page, in the order you selected them.
Can I reorder or remove images before converting?
Yes, use the "Remove" button next to each file in the list to adjust your selection before converting.
Is my image uploaded to a server?
No, the entire conversion happens locally in your browser using JavaScript.
Does the PDF keep the original image quality?
Yes, each image is embedded into its page at its original resolution.
