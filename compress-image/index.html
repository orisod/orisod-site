What is image compression?

Image compression reduces the file size of a photo by simplifying the data used to store it. This tool uses lossy compression, meaning some fine detail is discarded in exchange for a much smaller file — at moderate settings, the difference is usually invisible to the eye but the size savings are significant.

When should you use this tool?

  • Your photo is too large to attach to an email or upload to a form
  • You want your website to load faster by using smaller image files
  • You need to fit more photos into limited cloud storage
  • You're sending images over a slow or limited connection

Frequently asked questions

Is my image uploaded to a server?
No, compression happens entirely in your browser using the Canvas API — the file never leaves your device.
Does compressing reduce visible quality?
At quality settings above 65-70%, most people can't tell the difference visually, even though the file is significantly smaller. Lower settings trade more visible quality for smaller size.
What's the difference between compressing and resizing?
Compressing reduces file size while keeping the same pixel dimensions. Resizing changes the actual width and height. You can use both together for maximum size reduction.
What image formats are supported?
JPG, PNG, WEBP and other common formats your browser can read.
