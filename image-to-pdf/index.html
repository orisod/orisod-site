What does this tool do?

This tool combines any mix of JPG, PNG, and WEBP images into a single PDF document — one image per page, in the order you choose. It's the fastest way to turn a batch of photos, scanned pages, or screenshots into one shareable file.

When should you use it?

  • You scanned or photographed several pages of a document and need one PDF
  • You want to send a set of photos as a single, organized file instead of separate attachments
  • You're combining mixed formats (some JPG, some PNG) into a consistent document
  • You need to submit a portfolio, receipts, or ID photos as one file

Why use Orisod for this?

Everything runs locally in your browser using jsPDF — your images are never uploaded anywhere. You can reorder or remove images before converting, mix different formats freely, and each page keeps the original image's orientation and aspect ratio automatically.

Frequently asked questions

Can I mix JPG, PNG and WEBP in the same PDF?
Yes, you can select any combination of these formats and they'll all be combined into the same document.
Can I change the order of the pages?
Yes, images are added in the order you select them, and you can remove any of them before converting.
Is there a limit on how many images I can combine?
There's no fixed limit — you can add as many images as your browser's memory can comfortably handle.
Are my files uploaded to a server?
No. The entire process happens in your browser using JavaScript — nothing is sent anywhere.
